# SlutProdukt

Slutproduktet er et dashboard, der viser tre loves ændringer over tid (ændringer per år).

De tre love er helligdagsloven, udlændingenloven og færdselsloven.



## tech stack

Brug streamlit. Cache så meget som muligt i application. api er throttled så undgå at kalde unødigt. Foretræk stort kald fremfor flere små kald.
